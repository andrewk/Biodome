biodome-server
==============

[![Build Status](https://secure.travis-ci.org/andrewk/biodome-server.png?branch=master)](http://travis-ci.org/andrewk/biodome-server)

Node.js server for Device/Sensor network running on a Raspberry Pi

API
===
```
GET /devices
GET /devices/:id
PUT /devices/:id

GET /sensors
GET /sensors/:id
```
